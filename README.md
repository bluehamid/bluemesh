# 🌐 BlueMesh - VPN Network Solution

BlueMesh is a modern, user-friendly VPN network manager based on EasyTier.

## ✨ Features
- 🚀 Easy Installation
- 🎨 Modern UI
- 🔄 Auto-Update
- 🔒 WireGuard Support

## 🚀 Quick Install

```bash
bash <(curl -s https://raw.githubusercontent.com/bluehamid/bluemesh/main/install.sh)
```
